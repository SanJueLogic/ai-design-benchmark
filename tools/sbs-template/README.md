# SbS Voting Template

[English](#english) | [中文](#中文)

---

## English

This directory will contain an HTML template for creating Side-by-Side (SbS) blind voting sheets.

**Planned features:**
- Display N images side-by-side (one per product, randomly ordered, product labels hidden)
- 5-question voting form per task (Q1–Q5)
- Vote options: A / B / C / D / About the same / Not relevant
- Export results as CSV compatible with `tools/scoring.py`

**Status:** Planned for Phase 2. Contributions welcome — see [`docs/how-to-contribute.md`](../../docs/how-to-contribute.md).

---

## 中文

本目录将存放 Side-by-Side（SbS）盲测投票表的 HTML 模板。

**计划功能：**
- 并排展示 N 张图片（各产品随机排序，不显示产品名）
- 每条任务含 5 道投票题（Q1–Q5）
- 投票选项：A / B / C / D / 差不多 / 此题目与任务无关
- 导出 CSV 格式，与 `tools/scoring.py` 兼容

**状态：** 计划在阶段二实现。欢迎贡献 —— 详见 [`docs/how-to-contribute.md`](../../docs/how-to-contribute.md)。
