# Contributing

Thank you for your interest in contributing to AI Design Benchmark!

Full contribution guidelines are in [`docs/how-to-contribute.md`](docs/how-to-contribute.md).

Quick summary:
- New tasks: add entries to `dataset/v1.0/tasks.json` (include both `prompt` in Chinese and `prompt_en` in English)
- New scenes: open an Issue first to discuss scope
- Edit-type tasks: include reference image in `dataset/v1.0/reference-images/`
- All prompts must use the fictional brand "StarSelect / 星选" — no real brand names

Please read [`docs/how-to-contribute.md`](docs/how-to-contribute.md) before submitting a Pull Request.
